# V14.4 实验记录

## E12: V14.4 全量训练（无 shortcut, 无 LayerScale）

**配置**
- 架构: DAC-style 纯 sequential, 无 shortcut, 无 LayerScale
- latent_dim=128, strides=[2,4,4,4,4]=512, 43.1 Hz
- encoder_channels=[128,256,512,512,1024,1024], 121M params
- L1×10 + L2×5 + STFT×0.5 + Mel×0.1

**初期不稳定**: epoch 0-5 有严重 loss 爆炸（avg_G 到 2 亿），但模型自行恢复。

epoch 6 后完全稳定，后续 44 epoch 无任何异常。

## 对比总结

| 指标 | V14.2 (shortcut+LS) | V14.4 (无shortcut无LS) |
|------|---------------------|----------------------|
| Best SNR | **11.96 dB** | 11.36 dB |
| 频率盲区 | 2-6 kHz notch | **无盲区** |
| 初期稳定性 | 稳定 | epoch 0-5 爆炸后恢复 |
| 浅层梯度方向 | 冲突 (cos=-0.47) | **一致 (cos=0.29)** |
| 深层梯度衰减 | 有（shortcut 部分缓解） | 有（更严重） |

## 结论

V14.4 修复了频率盲区但牺牲了 0.6 dB SNR 和初期稳定性。
核心权衡：shortcut 提供梯度 highway 但引入 aliasing；无 shortcut 频率干净但深层梯度衰减。

## 下一步方向

1. **加判别器** — 在 V14.4 基础上加 MS-STFT discriminator，补偿高频细节
2. **数据增强** — 随机增益/pitch shift/加噪，提升数据利用率
3. **更长训练** — 200 epoch，当前 50 epoch SNR 还在缓慢上升
4. **梯度 highway** — 在 encoder 加 U-Net style skip connection（不用 pixel_unshuffle）
